## Features

- Search any songs
- Play/Pause song
- Fast Forward/ Rewind song
- Control Volume
- Recommended Artists


    
## Run Locally

Clone the project **Musico**

```bash
  git clone https://github.com/arindam369/Musico.git
```

Go to the project directory

```bash
  cd Musico
```

Install all the dependencies

```bash
  npm install
```

Start the Development Server

```bash
  npm run dev
```
